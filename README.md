Proyecto base para desarrollar la actividad JavaDoc de la Asignatura Entornos de Desarrollo
de 1º de DAW en ITT.

El proyecto consta de las siguientes clases:

- Suma:
	- Implementará: suma de dos números reales
					suma de dos números enteros
					suma de tres números reales
					suma con valor acumulado
- Resta:
	- Implementará: resta de dos números reales
					resta de dos números enteros
					resta de tres números reales
					resta con valor acumulado
- Producto:
	- Implementará: producto de dos números reales
					producto de dos números enteros
					producto de tres números reales
					potencia
- División
	- Implementará: división de dos números reales
					división de dos números enteros
					Inverso de un número real
					raíz de un número

Hay que desarrollar y documentar según se indica en el ejercicio.
